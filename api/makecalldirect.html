IraVoice API
IraVoice API

makecalldirect

Similar to makecall. But only to be used if you know exactly which cluster you want the call to go through.

Example usecase: Transfering / Bridging voicebot call to a human agent. When you trigger the first outbound call, the response JSON will have the cluster and call_uuid. Save the cluster value for later use.
If the voicebot requires to send call to human agent, call makecalldirect API with the cluster set to that of the original call.
The campaign_name will be the campaign exclusively created to send calls to human agents through suitable SIP trunk.
